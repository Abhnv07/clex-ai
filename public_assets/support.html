Frequently Asked Questions

How do I switch from raw OpenAI to CLEX? +
It's as simple as swapping the base URL to https://api.clex.in/v1 and passing in your CLEX API key. All endpoints are 100% compatible with the standard OpenAI SDK format.
Will you have access to my user data or prompts? +
We are zero-retention by default for prompt/response content in the proxy path. Requests are routed through CLEX with low-latency overhead.
What happens if a model drops offline? +
Our unified API automatically routes your request to a fallback cluster running identical weights (e.g. falling back from Meta's Llama cluster to NVIDIA's) when available. This reduces, but does not eliminate, the chance of user-visible disruption.
Can I get custom rate limits for production workloads? +
Yes. Please reach out to support@abhnv.in with your expected TPS (Tokens per Second) and concurrency requirements. We can provision dedicated VPC endpoints.
