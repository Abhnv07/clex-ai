# Clex.in - The Ultimate AI Wrapper API

**Live Website Preview:** [https://clex.in](https://clex.in)

Clex.in is a unified API proxy that allows developers to access a broad catalog of AI models through one OpenAI-compatible endpoint. Built for developers with a focus on speed, reliability, and ease of use.

## Features

- **Unified API:** Connect to DeepSeek, Llama, GPT-OSS, Qwen, Mistral, and many more.
- **Zero-Retention Privacy:** Prompt strings and completions are never logged or stored.
- **Ultra-low Latency:** Custom edge proxy infrastructure.
- **Dynamic Frontend:** Built with Tailwind CSS and advanced UI animations.
